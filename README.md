# ٹھیکیدار ڈیجیٹل - Labour Management System

یہ ایک ویب ایپ ہے جو مزدوروں کی حاضری، اجرت، خرچہ اور مالک ادائیگیوں کا ریکارڈ رکھنے کے لیے بنائی گئی ہے۔

## خصوصیات
- **لاگ ان**: او ٹی پی کے ذریعے سیکیور لاگ ان۔
- **مزدور اکاؤنٹ**: مزدوروں کو شامل کریں، حاضری لگائیں، خرچہ ریکارڈ کریں۔
- **مالک ادائیگیاں**: سائٹوں کی ادائیگیاں ریکارڈ کریں۔
- **خرچہ بک**: تمام خرچے ریکارڈ کریں۔
- **PDF ایکسپورٹ**: کسی بھی سیکشن کو PDF میں ڈاؤن لوڈ کریں۔
- **سرچ**: مزدوروں کو تلاش کریں۔
- **لوکل اسٹوریج**: ڈیٹا براؤزر میں محفوظ رہتا ہے۔

## استعمال
1. `index.html` کو براؤزر میں کھولیں۔
2. او ٹی پی حاصل کریں اور لاگ ان کریں۔
3. ٹیبز کا استعمال کرتے ہوئے سیکشنز میں جائیں۔
4. ڈیٹا شامل کریں اور منظم کریں۔

## ٹیکنالوجیز
- HTML, CSS, JavaScript
- SweetAlert2 برائے ڈائیلاگز
- html2pdf.js برائے PDF جنریشن

## نوٹ
ڈیٹا لوکل اسٹوریج میں محفوظ ہے، اس لیے براؤزر کلئیر کرنے پر ڈیٹا ختم ہو سکتا ہے۔
